A console-based **Bank Account Management System** developed in C++.  
The program simulates essential banking operations for both checking and savings accounts while demonstrating core object-oriented programming principles.

---

## 📌 Project Overview

This project implements a `BankAccount` class that manages:

- Checking Account Balance  
- Savings Account Balance  
- Customer Account Information  

The system allows users to perform essential banking operations such as deposits, withdrawals, balance inquiries, and fund transfers between accounts.

The objective of this project is to practice:

- Encapsulation  
- Constructors  
- Getters and Setters  
- Class-based Design  
- Input validation logic  

---

## 🚀 Features

- Create a bank account with customer name  
- Deposit funds into Checking or Savings  
- Withdraw funds with balance validation  
- Transfer funds from Checking to Savings  
- Display account details  
- Menu-driven console interface  

---

## 🏗️ Technical Highlights

- Fully encapsulated class design  
- Parameterized constructor  
- Controlled access to private data members  
- Validation of transactions (positive amounts & sufficient balance)  
- Separation of header and implementation files  
- Clean modular structure
